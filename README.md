To-Do List Web App

This is a simple, front-end To-Do List application built using HTML, CSS, and Vanilla JavaScript.

Features

- Add Tasks: Type a task and click "Add" or press Enter to add it to your list.
- Mark Complete:Click a task to toggle its completed status (line-through effect).
- Remove Tasks:Click "Remove" next to a task to delete it.
- Instant Updates: All actions update the UI instantly without reloading the page.

How to Run

1. Clone or download the repository.
2. Open index.html in your browser (using VS Code and live-server is recommended for auto-refresh).
3. Start managing your tasks!

Files

- index.html — Main HTML structure.
- style.css — Styles for layout and clarity.
- script.js — JavaScript for dynamic interactions.

---

Built as a beginner-friendly demo for task management and DOM manipulation.
